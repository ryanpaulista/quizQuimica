O que é a condesação?

Condensação é a transformação do líquido para o gás, como na evaporação.
Condensação é a passagem do estado líquido para o sólido, como na formação do gelo.
Condensação é a mudança do estado gasoso para o líquido.
Condensação é a mudança direta do sólido para o gás, como no gelo seco
